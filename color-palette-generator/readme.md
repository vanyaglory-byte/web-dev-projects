# 🎨 Color Palette Generator

A modern and interactive **Color Palette Generator** built using HTML, CSS, and JavaScript.

This application generates random color palettes and allows users to copy hex color codes directly to their clipboard with one click.

## ✨ Features

- 🎲 Generate 5 random colors instantly
- 📋 Copy hex code to clipboard
- 🎨 Modern gradient UI design
- 🖱️ Hover effects
- ⚡ Auto-generate palette on page load
- 📱 Responsive grid layout

- HTML5
- CSS3 (Grid & Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
